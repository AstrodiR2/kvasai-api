kvasai_demo.py
import openai
 
client = openai.OpenAI(
  base_url="https://kvasai.app/v1",
  api_key="kvs-..."
)
 
resp = client.chat.completions.create(
  model="llama-3.3-70b-versatile",
  messages=[{"role":"user",
             "content":"Привет!"}]
)
 
# ✓ 200 OK — 0.41s
print(resp.choices[0].message.content)

Преимущества

Зачем KvasAI?

Подключается туда, где уже стоит OpenAI

Меняешь один параметр — base_url — и всё готово. Open WebUI, LiteLLM, SillyTavern, продакшн-бекенд. Ни строчки рефакторинга.

API

Навсегда бесплатно

Никаких тарифов, никаких «бесплатного пробного периода». Просто работает.

Десятки моделей

Llama, Mistral, DeepSeek, Qwen, Gemma — в одном месте.

Быстро

Первый токен меньше секунды. Инфраструктура не дросселирует по тарифу.

Без регистрации карты

Только заявка и ключ. Ни платёжных данных, ни подписок.

Быстрый старт

Три шага до первого запроса

1

Открой бота

Перейди в @KvasAi_bot в Telegram и напиши /getkey — ключ выдаётся мгновенно.

Открыть бота →
2

Получи ключ

Бот сразу выдаст ключ в формате kvs-xxx.... Сохрани его — восстановить нельзя.

3

Подключи

Вставь Base URL и ключ в любой клиент с поддержкой OpenAI API — и всё работает.

Примеры использования

Base URL
https://web-production-293e6.up.railway.app/v1
Shell
curl -X POST https://web-production-293e6.up.railway.app/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"llama-3.3-70b-versatile","messages":[{"role":"user","content":"Привет!"}]}'
Python
import httpx

api_key = "YOUR_API_KEY"
base_url = "https://web-production-293e6.up.railway.app/v1"

response = httpx.post(
    f"{base_url}/chat/completions",
    headers={"Authorization": f"Bearer {api_key}"},
    json={
        "model": "llama-3.3-70b-versatile",
        "messages": [{"role": "user", "content": "Привет!"}]
    }
)
print(response.json())
JavaScript
const apiKey  = "YOUR_API_KEY";
const baseUrl = "https://web-production-293e6.up.railway.app/v1";

const res = await fetch(`${baseUrl}/chat/completions`, {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${apiKey}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "llama-3.3-70b-versatile",
    messages: [{ role: "user", content: "Привет!" }]
  })
});
console.log(await res.json());
